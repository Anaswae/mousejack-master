Cert VU#981271

Overview

A vulnerability has been discovered that allows keyboard HID packets to be 
injected into a computer equipped with a Dell KM632 wireless mouse.



Impact

This vulnerability allows an attacker to inject keystrokes into a computer 
equipped with a Dell KM632 wireless mouse. 



Affected Platforms

The following products have been tested and shown to be affected:

- Dell KM632 Wireless Mouse
- Dell USB dongle (USB ID 413c:2501)



Mitigation

Exploiting this vulnerability involves transmitting RF packets to a Dell USB 
dongle, which requires physical proximity to the target computer.
